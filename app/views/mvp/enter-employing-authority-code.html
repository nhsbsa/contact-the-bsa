An EA code is a 4-digit reference number unique to each employer who offers it's employee's access to the NHS Pension Scheme.

If you're an employer and need to find your EA code, email nhsbsa.pensionsemployers@nhsbsa.nhs.uk. You must include your:

  • name
  • position in the organisation
  • organisations name
  • organisations address.
